Capability Pattern: Imersao
Realiza-se uma coleta de informações baseada na observação do que pode ser necessário, para que seja identificado o que necessita de novo dentro de um ambiente (empresa). O intuito aqui é ter uma visão tanto interna, como externa de todo o local de aplicação do DT.

É feito uma coleta de dados com o cliente para compreender e definir qual o problema observado.
